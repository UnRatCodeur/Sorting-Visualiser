# Sorting-Visualiser

## About
This is a little program to display sorting algorithm with sounds, here a [video](https://www.youtube.com/watch?v=ZjFRBfVJaz0) of it.

![Solo](screenshots/solo.PNG)

![Create Account](screenshots/multi.PNG)

## Algorithms

* Bubble Sort
* Cocktail Sort
* Combo Sort
* Gnome Sort
* Heap Sort
* Insersion Sort
* Merge Sort
* Quick Sort
* Selection Sort
